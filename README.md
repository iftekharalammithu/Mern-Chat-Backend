## A real-time chat application built with MERN stack

### API

- `/api/auth/signup`,`/api/auth/login`,`/api/auth/logout`:
  - `POST`: Authenticate a user
- `/api/messages/message/:receiverid` , `/api/messages/:receiverid`:
  - `POST`: Send Message
  - `GET`: Get Message
- `/api/user/`:
  - `GET`: Get all user
